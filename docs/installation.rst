============
Installation
============


Stable release
--------------

To install apache-airflow-providers-teradata, run this command in your terminal:

.. code-block:: console

    $ pip install apache_airflow_providers_teradata

This is the preferred method to install apache-airflow-providers-teradata, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for apache-airflow-providers-teradata can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/flolas/apache_airflow_providers_teradata

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/flolas/apache_airflow_providers_teradata/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/flolas/apache_airflow_providers_teradata
.. _tarball: https://github.com/flolas/apache_airflow_providers_teradata/tarball/master
